# ОМИАС — Открытая музейная информационная автоматизированная система

Веб-приложение для автоматизации учёта музейных предметов. Разработано для Музея ГАИ с учётом требований Федерального закона № 54-ФЗ.

## Технологический стек

**Сервер:** Python 3.14, FastAPI 0.115, SQLAlchemy 2.0, PostgreSQL 15, Alembic, ReportLab

**Клиент:** React 19, TypeScript 6, Vite 8, Ant Design 6, React Router 7, Axios

## Требования

- Python 3.14+
- Node.js 18+
- PostgreSQL 15+

## Развёртывание

### 1. База данных

Создайте базу данных PostgreSQL:

```sql
CREATE DATABASE omias;
```

По умолчанию используется подключение `postgresql://postgres:postgres@localhost:5432/omias`. Для изменения задайте переменную окружения `DATABASE_URL`.

### 2. Серверная часть

```bash
cd backend
pip install -r requirements.txt
alembic upgrade head
python seed.py
uvicorn app.main:app --reload
```

Сервер запустится на http://localhost:8000. Документация API доступна по адресам `/docs` (Swagger UI) и `/redoc`.

### 3. Клиентская часть

```bash
cd frontend
npm install
npm run dev
```

Клиент запустится на http://localhost:5173.

### 4. Сборка для продакшена

```bash
cd frontend
npm run build
```

Собранные статические файлы из `dist/` можно раздавать через Nginx или другой веб-сервер.

После выполнения `seed.py` создаются учётные записи:

| Логин | Пароль | Роль |
|-------|--------|------|
| admin | admin123A | Администратор |
| keeper | keeper123A | Хранитель |
| researcher | research123A | Научный сотрудник |

Пароль необходимо сменить при первом входе.

## Структура проекта

```
backend/          — серверная часть (FastAPI)
  app/
    api/          — маршруты REST API
    core/         — конфигурация, БД, безопасность
    models/       — модели SQLAlchemy
    schemas/      — схемы Pydantic
    services/     — бизнес-логика
  alembic/        — миграции БД
  seed.py         — начальные данные
frontend/         — клиентская часть (React)
  src/
    api/          — HTTP-клиент
    components/   — общие компоненты
    contexts/     — контексты React
    pages/        — страницы приложения
diagrams/         — архитектурная и ER-диаграммы
```
